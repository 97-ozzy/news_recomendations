# News Recommendation System

Проект посвящён разработке рекомендательной системы для новостного контента.  
Задча - построить модель, которая предсказывает вероятность клика пользователя по новости на основе данных о новостях и истории пользовательских взаимодействий.
Цель работы - построить модель, которая попадет в топ 10 [Leaderboard](https://msnews.github.io/#leaderboard).

В проекте использовались данные из датасета [MIND-small](https://msnews.github.io/):

- `news.tsv` — информация о новостях: категория, подкатегория, заголовок, аннотация и другие признаки;
- `behaviors.tsv` — данные о поведении пользователей: история просмотров, показы новостей и клики.

В ходе работы был реализован полный пайплайн рекомендательной системы: загрузка и очистка данных, формирование обучающей выборки, построение baseline-модели, обучение итоговой модели и оценка качества рекомендаций.

## Что удалось достичь

В результате была построена модель, способная ранжировать новости и предсказывать вероятность пользовательского клика.

Качество модели оценивалось с помощью метрики AUC:

| Модель | AUC |
|---|---:|
| Baseline | 0.6285 |
| Mv1 | 0.6853 |
